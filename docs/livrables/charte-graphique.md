# Charte graphique - Vite & Gourmand

## 1. Positionnement visuel
Direction graphique: service traiteur premium, lisible, accessible, avec tonalite chaleureuse.

- Couleur secondaire: `#0f766e`
- Accent: `#f59e0b`
- Fond clair: `#f8fafc`

Note: verifier contraste final WCAG/RGAA pour chaque composant.

## 3. Typographie
- Police principale: system sans-serif (implementation actuelle)
- Hierarchie:
  - Titres H1/H2: graisse forte
  - Corps: regular
  - Labels et aides: regular/medium

## 4. Composants UI
- Boutons:
  - primaire (action principale),
  - secondaire (navigation),
  - danger (annulation/suppression).
- Formulaires:
  - labels explicites,
  - champs obligatoires identifies,
  - messages d'erreur visibles.
- Cartes menus:
  - titre, description, regime/theme, stock, CTA detail.
